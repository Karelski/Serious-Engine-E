Как работает эта система?

В данном каталоге находятся всего лишь 3 папки, куда вы можете положить GRO
архивы с ресурсами. Каждая из папок имеет свой приоритет загрузки.
Ниже перечислены папки сверху вниз по порядку загрузки.
Таким образом папка _Custom имеет самый низкий проритет.
--------------------------------------------------------------------------------
 _Custom\  - Папка для GRO с кастомными картами.
 Base\     - Папка для GRO со стандартными ресурсами.
 Coremods\ - Папка для GRO с заменами различных ресурсов.
